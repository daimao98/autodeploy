# 部署前准备

安装对应的包

npm install bluej

可用命令

bluej initVue 项目名 要部署的地址	# 拉取Vue项目

bluej inieEgg 项目名 要部署的地址	# 拉取egg项目

bluej toGit 项目名 git仓库地址 备注	# 上传项目至git仓库(备注也是必填项)

bluej run 项目名		# 进入Vue项目的开发 

bluej runegg 项目名	# 进入egg项目的开发
